# Generic Cloud Infrastructure Framework (IaC)

A professional AWS infrastructure built with Terraform. This framework deploys a highly secure VPC with Public/Private subnets, a Load Balancer, and a NAT Gateway.

## Features
- **High Availability:** Spans across multiple Availability Zones.
- **Security:** EC2 instances are hidden in Private Subnets.
- **Automation:** Auto-installs Apache and serves a dynamic metadata page.
- **Reusable:** Fully controlled via variables.

## How to use
1. Initialize Terraform: `terraform init`
2. Preview changes: `terraform plan`
3. Deploy: `terraform apply`
